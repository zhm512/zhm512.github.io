# DOM

# 1.1 DOM的概念

文档对象模型（Document Object Model，简称DOM），是W3C（万维网联盟）的标准，它允许程序和脚本动态地访问和更新文档的内容、结构和样式。

W3C DOM 标准被分为 3 个不同的部分

  • 核心 DOM - 定义了访问 HTML 和 XML 文档的标准

  • XML DOM - 定义了所有 XML 元素的对象和属性，以及访问它们的方法

  • HTML DOM - 定义了所有 HTML 元素的对象和属性，以及访问它们的方法。

    提示：我们重点学习核心DOM和HTML DOM

# 1.2 DOM节点

在DOM中，网页中所有的内容都被称为节点

  1. 整个文档是一个文档节点 （document）
  2. 每个 HTML 元素（标签）是元素节点 （element）
  3. HTML 元素内的文本是文本节点 （text）
  4. 每个 HTML 属性是属性节点 （attribute）
  5. 注释是注释节点（comment）

# 1.3 节点关系

  1. 根节点：一个HTML文档只有一个根，它就是HTML节点。
  2. 子节点：某一个节点的下级节点。
  3. 父节点：某一个节点的上级节点。
  4. 兄弟节点：两个节点同属于一个父节点。

# 1.4 DOM经常进行的操作

  • 获取页面元素
  • 动态创建、追加、插入、删除、克隆元素
  • 对元素进行操作(设置其属性或调用其方法)
  • 事件(什么时机做相应的操作)

# 1.5.2 HTML DOM 获取元素节点

  • 通过id获取元素节点（只找一个）

    var obj = document.getElementById('id名字')

  • 通过标签名获取元素节点（返回伪数组）

    var obj = document.getElementsByTagName('标签名')

  • 通过类名获取元素节点（返回伪数组）

    var obj = document.getElementsByClassName('类名')

  • 通过选择器获取元素节点

    var obj = document.querySelector('选择器')（只找一个）

var obj = document.querySelectorAll('选择器')（返回伪数组）

  • 通过name属性获取表单元素节点

    var input = document.getElementByName('表单元素名');
或者
var inputs = document.getElementsByName('表单元素名');

# 1.6 获取节点类型、节点名称、节点值

  • 获取节点类型

    语法：dom节点.nodeType
    节点类型 nodeType
    元素 1
    属性 2
    文本 3
    注释 8
    文档 9

  • 获取节点名称

    语法：dom节点.nodeName
    节点类型 nodeName
    元素 大写标签名
    属性 属性名
    文本 #text
    注释 #comment
    文档 #document
    提示：
元素节点.attributes：获取元素节点所有的行内属性

childNodes和children的区别，childNodes获取的是所有子节点，children获取的是所有子元素

  • 获取节点值

    语法：dom节点.nodeValue

提示：只有文本节点、属性节点有节点值
文本节点的 nodeValue 是文本本身 
属性节点的 nodeValue 是属性值

# 1.7. 事件基础

# 1.7.1. 事件概述

JavaScript 使我们有能力创建动态页面，而事件是可以被 JavaScript 侦测到的行为。

简单理解： 触发--- 响应机制。

​ 网页中的每个元素都可以产生某些可以触发 JavaScript 的事件，例如，我们可以在用户点击某按钮时产生一个 事件，然后去执行某些操作。

# 1.7.2. 事件三要素

  • 事件源（谁）：触发事件的元素
  • 事件类型（什么事件）： 例如 click 点击事件
  • 事件处理程序（做啥）：事件触发后要执行的代码(函数形式)，事件处理函数

案例代码

<body>
    <button id="btn">唐伯虎</button>
    <script>
        // 点击一个按钮，弹出对话框
        // 1. 事件是有三部分组成  事件源  事件类型  事件处理程序   我们也称为事件三要素
        //(1) 事件源 事件被触发的对象   谁  按钮
        var btn = document.getElementById('btn');
        //(2) 事件类型  如何触发 什么事件 比如鼠标点击(onclick) 还是鼠标经过 还是键盘按下
        //(3) 事件处理程序  通过一个函数赋值的方式 完成
        btn.onclick = function() {
            alert('点秋香');
        }
    </script>
</body>
