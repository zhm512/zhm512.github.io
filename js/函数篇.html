    # 函数篇

    1. 函数的声明及调用

      • 关键字 function 函数名(){//函数体}

      • 函数表达式 var 变量名 = function(){//函数体}

        区别：关键字创建函数会把函数整体提前，所以在哪里调用都可以；函数表达式创建函数只是把变量声明提前，所以在赋值之前调用会报错，只能在赋值之后调用

      • 函数的调用

    2. 函数的参数

      • 形参：是函数创建的时候()里放的变量
      • 实参：是函数调用的时候传入()中的实际数据

    3. 函数的arguments是什么

      • Arguments是个类似数组但不是数组的对象，说他类似数组是因为其具备数组相同的访问性质及方式，能够由arguments[n]来访问对应的单个参数的值，并拥有数组长度属性length。还有就是arguments对象存储的是实际 传递给函数的参数，而不局限于函数声明所定义的参数列表，而且不能显式创建 arguments 对象。
