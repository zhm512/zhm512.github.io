# 栈存储和堆存储

  • 基本数据类型 存在栈内存中
  • 引用数据类型的名字也存在栈内存
  • 对象的浅拷贝和深拷贝
  • 对象的浅拷贝：对象的浅拷贝采用的引用地址的传递，没有在堆内存中开辟新空间，所以修改其中一个对象另一对象也跟着修改
  • 浅拷贝的代码演示
var obj = {
	name :'zhangsan',
	age: 11
}

var obj1 = obj;

对象的深拷贝：对象的深拷贝在堆内存中开辟新空间，通过for in循环把其中一个对象的内容拷贝到新对象中，所有修改其中一个另一个不会跟着修改

var obj2 = {
	name: '李四'，
	age： 45，
	height: 190
}

var obj3 = {}

for(var j in obj2){
	obj3[j] = obj[j]
}
